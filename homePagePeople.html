Week 1: Creating a new HTML website.

Published on October 8th, 2020

In my first lab session, I learned to -

  1. Set up a folder for my website on my Brighton Domains workspace
  2. Link two pages together with a navigation menu using a relative link
  3. Create links to external websites using an absolute link
  4. Use h1 and h2 tags
  5. Edit the page title
  6. Validate HTML using the W3C validator

HTML (the Hypertext Markup Language) and CSS (Cascading Style Sheets) are two of the core technologies for building Web pages.
